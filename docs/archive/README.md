# Parkly Documentation Archive

This folder contains superseded or historical documentation artifacts.

## Archive Structure

- superseded project snapshots such as older specs and plans
- patch notes and migration notes from prior delivery phases
- historical reference notes that are no longer canonical
- retired `.docx` deliverables that should not be treated as the current approved snapshot

## Archive Rule

Nothing in this folder should be linked as the current source of truth from operational docs unless it is explicitly called out as historical context.
