# Frontend mobile-review evidence

Thư mục này dùng để lưu evidence thật sau mỗi vòng smoke cho mobile capture và review workflow.

## Layout

- `manual-run-sheet.md`: checklist thao tác cho người test
- `bug-closure-notes.md`: đóng bug theo từng mục
- `round-1/`: happy path
- `round-2/`: fresh pair token
- `round-3/`: edit secret giữa chừng

Mỗi round nên có:

- screenshot pair origin
- screenshot mobile context summary
- screenshot heartbeat status
- screenshot capture status
- screenshot session/review detail
- screenshot terminal lock nếu có
- file text ngắn ghi requestId và kết quả thực tế
