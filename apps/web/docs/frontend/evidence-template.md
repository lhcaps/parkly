# Evidence template

Bộ evidence tối thiểu khi bàn giao FE:
- `latest-smoke.json`
- build log terminal
- 6 đến 10 screenshot của route chính
- manual QA signoff đã điền tên người kiểm tra và ngày kiểm tra

## Screenshot set tối thiểu
1. Login
2. Overview
3. Run Lane
4. Review Queue với detail mở
5. Session History hoặc Audit Viewer
6. Sync Outbox
7. Mobile Pairing hoặc Mobile Capture
8. Một state degraded / stale / forbidden nếu tái hiện được
