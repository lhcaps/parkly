Quickstart

Mục tiêu: demo “app-like” bằng UI chạy trực tiếp (không build).
1) copy .env.example .env 2) pnpm install 3) pnpm e2e 4) pnpm api:dev Mở: http://127.0.0.1:3000/ui/
Nếu bật API_AUTH_MODE=ON, hãy paste token đúng role ở khung “Auth bar” bên phải.

Auth bar (demo)

SSE/EventSource không set được header, nên UI sẽ tự gắn ?token=... (cần API_ALLOW_QUERY_TOKEN=ON).
